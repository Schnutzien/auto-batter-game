## Game Description

Auto Batter Game is a clicker/idle game where you earn points by clicking a button and purchase upgrades to automate your clicking. The more upgrades you buy, the more points you'll earn automatically!

## How to Play

1. Click the "CLICK!" button to earn points
2. Purchase upgrades using your earned points:
   - **Auto Clicker** (10 points) - Earns 1 point per second
   - **Power Clicker** (100 points) - Earns 10 points per second
   - **Super Clicker** (1000 points) - Earns 100 points per second
3. Watch your score grow automatically!
4. Unlock more upgrades and progress through the game
